Utilizando <h1> até <h6>

Voltar ao índice

As heading tags podem ser trabalhadas em uma escala de 1 a 6 (do H1 ao H6), a fim de estabelecer uma organização hierárquica das informações. A tag H1 é a mais importante da página e deve ser atribuída ao título principal do conteúdo. As tags H2 são vinculadas aos subtítulos, as tags H3 a outros trechos de destaque (mas com menor relevância que as tags H2), e assim por diante. Ao contrário do que para muitos pode parecer, é completamente dispensável a utilização de todas as heading tags em um texto.

Utilizando a tag <h1>

A tag H1 não só é a tag mais importante da página, como também a única indispensável. Isso porque, dependendo do conteúdo, pode não haver necessidade de realizar desdobramentos para subtítulos. No entanto, um título principal sempre deve existir.
Por ser a tag de cabeçalho, que informa aos mecanismos de busca sobre o que se trata determinada página, ela exerce influência no ranqueamento orgânico. Por todas essas razões, essa tag merece mais a sua atenção. Algumas dicas:
Utilize apenas uma tag H1 por página
Como já vimos anteriormente, a tag H1 é a mais importante da página e deve ser atribuída a seu título principal. Isso significa que deve haver apenas uma H1 por página para que o seu poder não seja diluído.
Atribua uma H1 exclusiva para cada página do site
É muito importante que você não replique o mesmo conteúdo da tag H1 para outras páginas, ou seja, cada página deve possuir uma tag H1 exclusiva. Até porque cada página de seu site deve ter um foco diferente e, portanto, palavras-chave diferentes. O conteúdo duplicado pode gerar problemas para o seu desempenho.
Insira a palavra-chave principal na H1
Sabendo que a tag H1 informa aos buscadores sobre o que se trata a sua página e que esta pode influenciar na sua classificação, subentende-se que a H1 deve conter, preferencialmente, a palavra-chave principal da página. Ademais, há espaço para explorar outros termos relevantes, desde que ocorra de uma maneira coerente e natural.

Utilizando as tags <h2> a <h6>

Normalmente utilizamos a tag h2 para índices ou inícios de capítulos, a questão que envolve é a hierarquia de importância. Vimos anteriormente que o h1 é a palavra chave da página, então só pode haver uma por página. As tags h2, portanto, são subdivisões da página, assim como a tag h3 é uma subdivisão do conteúdo dado por h2 e por aí vai.
Quando você abre esta presente página o título dele "Desafio 1 - Revisão do conteúdo" está em h1 enquanto o Índice e o tópico "Utilizando tags h1 a h6" estão em h2. Os tópicos específicos "Utilizando a tag h1" e "Utilizando as tags h2 a h6" (presente tópico que você está lendo) estão em h3 porque são subdivisões de "Utilizando tags h1 a h6".

Utilizando <p>

Nós utilizamos p para começar um parágrafo, esse texto que você está lendo é exatamente um parágrafo escrito com o a tag p no código HTML.

Ao iniciar outro parágrafo, como este, é dado o espaço de uma linha.

Utilizando <Mark>

O elemento <mark> é usado para destacar partes do texto. Nessa frase a palavra destacar está sobre efeito do <mark>, no código HTML aparece como <mark> destacar <mark> .

Utilizando <Small> e <a>

O elemento < small > torna as palavras na forma pequena como se apresenta.
O elemento <a> é usado para criar link que enviam o usuário para outra página da web ou para outras seções da mesma página.
Por exemplo ao escrever o código a seguir :

<a> href="https://www.youtube.com/" target="_blank"> Youtube </a>

criamos no HTML o link abaixo :

Youtube

Onde :

  • href=" argumento " recebe como argumento o link do site.
  • target="_blank" significa que ao clicar o site abrirá e outra aba
Uma aplicação de < small > e < a > foi na utilização de link de retorno para o índice :
Voltar ao índice
Que se encontra nos subtítulos da página. Abaixo está o código:

< small > < a href="#indice"> Voltar ao índice </a > < /small >



Utilizando < i > , < u > e < strong >

A tag < i > transforma os caracteres em itálico . Veja o código abaixo em HTML:

< i > Essa frase está em itálico. < /i >

Aparece na tela do site da seguinte forma :

Essa frase está em itálico.

A tag < u > adiciona um sublinhado o significado do "u" vem do inglês underline. Veja o código abaixo em HTML:

< u > Essa frase está em itálico. < /u >

Aparece na tela do site da seguinte forma :

Essa frase está sublinhada.

A tag < strong > coloca os termos em negrito o significado de "strong" vem do inglês significa forte. Veja o código abaixo em HTML:

< strong > Essa frase está em itálico. < /strong >

Aparece na tela do site da seguinte forma :

Essa frase está sublinhada.

Utilizando <ol>, <ul> ,<li>

Utilizamos <ol> juntamente com <li> para criar uma lista ordenada. <ol> vem do inglês ordened list e <li> ve do inglês list element.


O código HTML fica dessa forma :

<ol>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ol>

No site aparece :
  1. Item 1
  2. Item 2
  3. Item 3

Utilizamos <ul> juntamente com <li> para criar uma lista ordenada. <ul> vem do inglês unordened list.


O código HTML fica dessa forma :

<ul>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ul>

No site aparece :
  • Item 1
  • Item 2
  • Item 3
